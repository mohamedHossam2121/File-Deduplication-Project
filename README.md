file deduplication system that identifies and removes duplicate files to optimize storage usage

 How to Use it
- Step 1: run generate_files.py which creates the number of files we test the project on.
- Step 2: select one of the algorithms file to run and test the algorithm, choose from(recursive.py,greedy.py,divide_and_conquer.py)
- Step 3: run comparison.py to test the execution time for each algorithm and compare the execution time
